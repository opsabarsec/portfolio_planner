# EU AI Jobs and Skills Module

## Goal
Translate hiring demand into concrete portfolio choices for the European market.

## Search focus
- AI engineer jobs Europe
- LLM engineer jobs France Germany Netherlands
- Agentic AI jobs Europe
- MLOps and AI platform roles
- RAG, evaluation, observability, governance, and inference infrastructure requirements

## Extraction targets
- Repeated stack keywords
- Architecture patterns
- Compliance or language requirements
- Cloud and deployment expectations
- Industries hiring most actively

## Output format
- Skill cluster
- Frequency signal
- Example job contexts
- Portfolio evidence to build
